Projects

Yak

I built Yak, a clone of Slack. It has a Rails backend that takes advantage of ActionCable, a recent addition to rails that implements WebSockets, to provide real-time live-updating chat. React paired with Redux is used on the frontend to create a snappy and consistent user interface while being maintainable and extensible by the developer.

Figgie

I built Figgie, a card game designed by Jane Street to be poker for the stock market. It takes advantage of a Web Animation framework that I built to abstract over CSS transitions. All I do is implement game logic by moving HTML elements around, and describe their appearance with standard CSS, and the framework seamlessly animates everything in between.
